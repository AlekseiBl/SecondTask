## Installation

```
$ git clone https://github.com/AlekseiBl/SecondTask.git
$ cd webpackdir
$ npm install
$ npm run build
```

## Run


```
$ git clone https://github.com/AlekseiBl/SecondTask.git
$ cd webpackdir
$ npm install 
$ npm run start:dev
```
